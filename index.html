Linear Regression

Statistical model depicting the relationship between Temperature and Humidity.

Blue Line

It is the true fitting line which represents the true or the exact value of y for each possible value of x. This is showing the value of Temperature for each value of Humidity

Pink Line

It is the logarithm fit of the data which makes the more skewed data, less skewed. It makes the data more interpretable.

Stream Lines

The relationship between two variables from linear regression the flow of the data can be understood from streamlines. It represents the flow of the data points which is integrated with velocities to simulate the flow of the path of data points







Grouping together of neighboring data points is done by streamlines with which velocities are integrated to imitate the path which a particle should follow if placed in a flow.

Lets See What these Transitions are:


The transitions are Line chart, Stacked area chart, Overlapping chart, Grouped Bar chart, Bar chart, Donut chart

  • Line Chart: Temperature and Dew point seem to be having a directly proportional relationship as whenever the temperature is increasing, the dew point is also increasing
  • Stacked area Chart: It is showing the relationship between temperature and dew point with intensities as it is prominent that if the temperature is decreasing by 10 units the dew point is decreasing by 6 units approximately.
  • Overlapping Chart: At every time frame, it shows the intersection point of at least 2 values from temperature, humidity and dew point where they will share the exact same value.
  • Group bars Chart: These bars are demonstrating the values for dew point and humidity at all time frames. The orange line represents the humidity, blue line for temperature and green one is for the dew point.
  • Bar Chart: It is a cumulative percentage for the complete dataset and shows that, on an average, the temperature is always higher than humidity and dew point.
  • Donut Chart: It is a space-efficient chart which remedies the arc reading problem faced in pie-chart by de-emphasizing the use of an area. It is displaying the ratio of temperature, humidity and dew point on an average.
